## Install

1. Open **Extensions** sidebar in VS Code/Cursor
2. Search for `Farm to Table`
3. Click **Install**
4. Open Command Palette (`Cmd+K, Cmd+T` or `Ctrl+K, Ctrl+T`)
5. Select **Farm to Table Dark** or **Farm to Table Light**
